# 💗 Ourverse

> *A private little universe for two people.*

A romantic interactive web application designed for a couple to exchange digital coupons and celebrate their relationship. Built with love using Next.js, TailwindCSS, and Framer Motion.

---

## ✨ Features

- **Magical Landing Page** — Animated bloom background, live relationship countdown timer, and rotating love affirmations
- **Romantic Light Switch Toggle** — Switch between Create Mode and Explore Mode with smooth animations
- **Coupon Exchange System** — Create beautiful collectible coupon cards for your partner
- **Coupons Page** — View, filter, and redeem coupons with confetti celebrations
- **Floating Particles** — Stars and hearts gently float across the screen
- **Glass Morphism Design** — Dark romantic theme with glowing gradients


## 🛠 Tech Stack

- **Next.js 14** — App Router
- **TailwindCSS** — Styling
- **Framer Motion** — Animations
- **LocalStorage** — Coupon persistence
- **canvas-confetti** — Redemption celebrations
